Appium Q&A - Session Notes

❓ How do I find the package and activity name in Appium using Java?
Use ADB: 
adb shell dumpsys activity activities | findstr "ResumedActivity"
Example output: com.example.app/.MainActivity Use in Appium: 
caps.setCapability("appPackage", "com.example.app");
caps.setCapability("appActivity", "com.example.app.MainActivity");
❓ Getting `'grep' is not recognized` error?
You're on Windows. Use findstr instead of grep: 
adb shell dumpsys activity activities | findstr "ResumedActivity"
❓ My output shows: com.vyom.okasmax/.boarding.OABoardingActivity — is this correct?
Yes, that's the current resumed activity. But if Appium can't start it directly, it may not be a launcher activity.
